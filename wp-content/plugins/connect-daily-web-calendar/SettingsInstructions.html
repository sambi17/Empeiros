Settings Information

Calendar URL This is the URL to your Connect Daily Calendar. If you don't already have a calendar, select the radio group option Create a new Connect Daily Calendar system.

User Name and Password are only required if the public cannot view the calendar, or you want to display a calendar the default public user does not have permission to.

Date-Time Format is used when an event with a date and time is displayed. Date Format and Time Format are individually controlled by Settings | General.

Skip Resource Types By default, all resources assigned to an event are displayed. You can

  1. Leave this blank to list all resources.
  2. Specify a comma-separated list of resource type names that should not be displayed
  3. Put an asterisk (*) in to skip all resources.

Disable Caching By default, the plugin caches generated data for seconds. Caching the data makes your web site go faster and gives your users a better experience. If you edit your calendar, changes might not show up immediately. You can temporarily disable this by checking the box and saving.
