Overview

The Connect Daily plug-in provides an interface to that Connect Daily cloud calendar system. Connect Daily is incredibly powerful and full-featured. It includes:

  • Multi-User with Configurable Security
  • Event Approvals
  • SMS or EMail Notifications and Reminders
  • Resource Management
  • Social Networking Support
  • Mobile Device Support
  • Data Import/Export
    • iCalendar, RSS, JSON, and CSV Export
    • iCalendar, RSS, and CSV Import. Supports recurring imports from remote urls.
  • Time Zone Support
  • Outstanding WordPress Support
  • High Performance - Supports 10's of thousands of events.
  • See a complete list of features.

For complete information about Connect Daily, visit our web site.

This plugin provides

  • Responsive Full-Sized Calendar
  • Responsive Public Event Add Shortcode
  • Mini-Calendar Widget
  • Responsive Detailed List of Events Shortcode
  • Responsive Simple List of Events Widget and Shortcode
  • iCalendar Widget and Shortcode
  • IFRAME Shortcodes for other calendar formats into your design.

In addition to our best-in-class feature set, we offer outstanding support which includes unlimited telephone support to paid customers.
